配置github

  1. 登录Github账号，没有自己注册一个

  2. 点击右上角头像->setting

  3. 选择SSH and GPG keys

  4. 点击New SSH Key
    标题随便，KEY就是刚刚复制的那个，保存即可

  1. 验证配置 出现如下提示，表示配置成功
    Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.
